## ⚙️ `crontab -e  # Automate everything`

<details>
<summary><b>👾 Setup: Snake Animation (.github/workflows/snake.yml)</b></summary>

```yaml
name: Snake 🐍
on:
  schedule: [{ cron: "0 */12 * * *" }]
  workflow_dispatch:
jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

</details>

<details>
<summary><b>🧊 Setup: 3D Contribution Graph (.github/workflows/profile-3d.yml)</b></summary>

```yaml
name: 3D Contrib 🌐
on:
  schedule: [{ cron: "0 18 * * *" }]
  workflow_dispatch:
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: yoshi389111/github-profile-3d-contrib@0.7.1
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          USERNAME: ${{ github.repository_owner }}
      - run: |
          git config user.email "action@github.com"
          git config user.name "GitHub Action"
          git add -A && git commit -m "🧊 3D contrib [auto]" || true
          git push
```

</details>

<details>
<summary><b>🎮 Setup: Space Shooter Contribution GIF (.github/workflows/space-shooter.yml)</b></summary>

```yaml
name: Space Shooter 🎮
on:
  schedule: [{ cron: "0 0 * * *" }]
  workflow_dispatch:
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Generate Space Shooter
        uses: czl9707/gh-space-shooter@v1
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          username: ${{ github.repository_owner }}
          output_file: dist/github-space-shooter.gif
      - name: Deploy to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

> **Note:** Check the [czl9707/gh-space-shooter](https://github.com/czl9707/gh-space-shooter) repo for the exact action syntax — update above if needed.

</details>
